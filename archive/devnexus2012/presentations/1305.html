Java-Powered Cross-Platform Mobile Applications with PhoneGap

Skill Level:
Abstract

Are you ready to bring your web development skills to natively installed mobile applications on multiple platforms? In this session get ready for a deep dive into HTML & JavaScript powered cross-platform applications using PhoneGap (Apache Callback), powered by enterprise Java backend systems.
