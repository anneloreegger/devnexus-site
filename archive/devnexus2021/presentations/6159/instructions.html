Liquid Software: DevOps Tools for Developers

Attendees will need the following:

JFrog Platform Account - Free tier

Sign up for the JFrog Platform Free Tier http://jfrog.co/devnexus-workshop

IMPORTANT: We will be using the JFrog Pipelines feature during this workshop. Please choose ‘Cloud’ and either ‘AWS’ or ‘Microsoft Azure’ as your cloud provider in order to get JFrog Pipelines support.

IDE that supports Java, Maven and Integration with JFrog Xray

IntelliJ is recommended and the free version will suffice for this workshop: https://www.jetbrains.com/idea/download/Visual Studio Code and Eclipse will also work if you are more comfortable with these. Instructions for IDE integration with JFrog can be found here: 

https://www.jfrog.com/confluence/display/JFROG/IDE+Integration 

Java 8 or higher

Any JDK will work.

Maven 3.3.1 or higher (3.6.3 is recommended)

Download: https://maven.apache.org/download.cgi

Installation Instructions: https://maven.apache.org/install.html

GitHub account 

Sign up at github.com if you don’t already have an account.

Docker

Docker Desktop is recommended: https://www.docker.com/products/docker-desktop

Google Cloud Platform Account

The 90-day, $300 free trial will suffice for this workshop: https://cloud.google.com/free

Registrants are more than welcome to email me with any setup questions prior to the workshop. melissam@jfrog.com
