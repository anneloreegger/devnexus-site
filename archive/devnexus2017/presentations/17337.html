Presentation Link
Abstract

This session focuses on what is data and how data is used to inform the design process for new products, features, and the overall user experience. There is a lot of buzz around data; however, having lots of data alone is not useful. As developers, we play an important role in how data is captured / extracted / obtained, how data is analyzed and merged across multiple data sources, and how the findings are applied to improve designs.
