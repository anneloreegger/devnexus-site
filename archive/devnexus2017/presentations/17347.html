Adding Dynamic Back to Static Site Generators

Track: Web
Skill Level: Beginner
Room: Room A405
Time Slot: Thu 2/23, 1:00 PM
Tags: static sites
Presentation Link
Abstract

Moving to a Static Site Generator (SSG) can be a great way to simplify your production needs for your web site. But once you’ve done that, how do you go about adding dynamic aspects back? In this talk I’ll talk about how to things like contact forms, comments, search, and calendars back into a SSG.
