Abstract

For years, developers have tried to find an easier way to create mobile apps; they discovered hybrid mobile apps but soon found out that hybrid wasnâ€™t ready for prime-time. Developers encountered problems with speed, building good-looking UIs, and a plethora of hardware differences that forced them to go back to the land of native development. AngularJS changed the way developers built apps for the web, and now Ionic has changed the way developers build hybrid apps on mobile.

Easy-to-use, beautiful, and ready to perform, Ionic is an open source framework made for hybrid apps. In this talk, see how to use AngularJS and how to supercharge it with Ionic to create an SDK for the web.
